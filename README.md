# Python File Organizer

Automatically organizes files into folders based on file extension.

## Features

* Remembers extension → folder mappings
* Automatically organizes common file types
* Works on any directory
* Skips system files like `.DS_Store`

## Usage

Navigate to desired folder on terminal and run
```bash
python main.py ~/Downloads
```

Example result:

```
Downloads/
    pdf/
    images/
    videos/
```
